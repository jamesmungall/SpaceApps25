Make your own

We have made the beginnings of a python package which would allow the user to create their own videos

Who is it for?

The python package could be used by a teacher or anyone presenting an image of space. They can create a video by inputting the URL of the image and the right ascension and declination of the object.

These scripts could also be extended and improved by anyone wanting to develop this further.

How to do it

Create a csv file of the following format:

name url ra dec text1 text2 text3
Horsehead_Nebula https://images-assets.nasa.gov/image/PIA16008/PIA16008~large.jpg 85.2458 -02.4583 "Hubble Sees a Horsehead of a Different Color" "Backlit wisps along the Horsehead Nebula upper ridge are being illuminated by Sigma Orionis, a young five-star system just off the top of this image from the Hubble Space Telescope." "Taken April 19, 2013, Producer JPL, Size 1837x1920px, PIA Number PIA16008"

Download our python scripts from GitHub at https://github.com/jamesmungall/SpaceApps25.git
