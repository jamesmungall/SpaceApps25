Hubble Sky Viewer!

We have created scripts in Python which run on the Gaia Sky application to help visualise the Hubble Space Telescope's high resolution images.

Our python scripts create a walkthough from earth, zooming in on low resolution textures and overlaying them with much higher resolution images taken from NASA's Hubble image archive

This gives you a sense of perspective as to where these amazing images fit into the cosmos around us allowing the large image files from Hubble to be fully appreciated. We have included the detailed captions provided by NASA to help give more context for what you are seeing.

What resources have we made?

1. A series of videos

2. The beginnings of a python package which would allow the user to create their own videos

Who is it for?

The videos could be used directly by primary school aged children. For example, a teacher may show these in class as an introduction to astronomy. The detailed captions, taken from the NASA website, give some scientific detail about the images.

The python package could be used by a teacher or anyone presenting an image of space. They can create a video by inputting the URL of the image and the right ascension and declination of the object.

These scripts could also be extended and improved by anyone wanting to develop this further.
