What have we done?

Before running Gaia Sky we have done the following in Python:

  • Read the name, image url, right ascension, declination and text from the csv file
  • Downloaded the image from the url
  • Resized image
  • Added border

Then, again using Python, we have programmed Gaia Sky to:

  • Go to earth
  • Rotate to the correct right ascension
  • Tilt to the declination
  • Zoom in
  • Dixplay image
  • Display text
